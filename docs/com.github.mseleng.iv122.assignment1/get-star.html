fun getStar(steps: Int, stepWidth: Int, lineStyle: Style = defaultStyle): SVG

Prints the "tangent" star. The result image can be seen in the file 'outputs/star.png'

Parameters

steps - the number of lines of the half of the horizontal/vertical center line (the size of the result file is (2*steps*stepWidth)

stepWidth - the size of the gap between each step (how far from each other the points are)

lineStyle - the style of the line. Default: defaultStyle
